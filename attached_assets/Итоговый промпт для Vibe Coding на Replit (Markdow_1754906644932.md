## Цель

Создать современный адаптивный сайт-визитку/каталог для АЛЬТА СЛЭБ (стеновые и потолочные SPC-панели) с визуальной ссылкой на стиль arch-skin.ru: минимализм, крупная типографика, много «воздуха», аккуратные hover-эффекты и стандартная вертикальная прокрутка.

## Дизайн и референс

- Референс: минималистичный премиальный стиль arch-skin.ru — крупные заголовки, светлый фон, сетка карточек, простая навигация.
- Макро-принципы: чистые сетки, фокус на фото/текстурах, аккуратные анимации по hover, без параллаксов.
- Цветовая схема (обязательно использовать как CSS-переменные):

```css
:root{
  --color-primary:#2d2c2c;   /* Заголовки, навигация */
  --color-accent:#c09595;    /* CTA, премиум-метки */
  --color-secondary:#715a58; /* Текст, активные/hover */
  --color-text:#909090;      /* Вторичные тексты */
  --color-border:#848484;    /* Разделители/бордеры */
  --color-bg:#ffffff;        /* Фон */
}
```

- Типографика:
    - H1: 3rem, 700, letter-spacing:2px, color: var(--color-primary)
    - H2: 2rem, 700
    - Текст: 1rem–1.125rem, color: var(--color-secondary)
    - Вторичный текст/метки: color: var(--color-text)
- Карточки:
    - Фон: \#fff, border:1px solid var(--color-border), border-radius:8px
    - Hover: тень + border-color: var(--color-secondary); плавность 200–250ms
- CTA:
    - Кнопки: фон var(--color-accent), текст \#fff; hover фон var(--color-secondary)
